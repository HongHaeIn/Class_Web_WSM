💙 wsm 배우기 💙

  • selector
    1. 태그
    2. 클래스 .
    3. 아이디 #
    4. 자식선택자 p>h1
    5. 자손선택자 p h1
    6. 인접형제선택자 h1 + p: h1의 첫번째 형제 p에 적용
    7. 연결형제선택자 h1~p: h1의 모든 형제 p에 적용
    8. 속성선택자
      • [속성]: 지정한 속성을 요소를 찾아 적용
      • [속성=값]: 속성=값으로 일치하는 요소를 찾아 적용
      • [속성~=값]: 해당값이 포함되어 있는 요소를 찾아 적용
      • [속성|=값]: 특정값이 포함된 요소를 찾아 적용
      • [속성^=값]: 특정값으로 시작하는 요소를 찾아 적용
      • [속성$=값]: 특정값으로 끝나는 요소를 찾아 적용
  • 가상선택자
    • -마크업상에 드러나지 않음 (태그자체가 없음)
    • -사용자의 동작, 그때의 환경에 따라서 요소에 적용하는 선택자
    1. a:link (검정,밑줄없애게)
    2. a:visited (검정,밑줄없애게)
    3. a:hover 마우스 갖다 댔을 때(밑줄, 파란색)
    4. a:active 클릭했을 때
    5. :focus
    6. :nth-child(n) / :nth-of-type(n) / :first-letter
      nth-child(2n+1) /nth-child(2n)
      • p:nth-child(2)
        ==> 부모의 2번째 자식이면서, 그 자식이 p태그이면 적용
      • p:nth-of-type(2)
        ==> p태그중에서 2번째 p태그에 적용
  • position
    1. 정적배치(static position)
      • 기본으로 배치
      • top, left, bottom, right 가 적용이 안됨
    2. 상대배치(relative position):
      • top, left, bottom, righ가 적용이 됨
      • 기준위치: 원래 정적배치 위치가 됨
    3. 절대배치(absolute position):
      • top, left, bottom, righ가 적용이 됨
      • 기준위치 : 브라우져 맨 최 상단
    4. 고정배치(fixed position);
      • top, left, bottom, righ가 적용이 됨
      • 브라우저 상에서 고정
      • 부모는 relative, 자식은 absolute로 적용
  • transition, transform
    1. transition속성
      • transition: 변하고자 하는 값들, 시간
      • transition: width, 5s;
    2. transform
      • 회전( rotation)
      • 크기 확대 축소(scaling)
      • 평행이동(translation)
      • 비틀기(skew)
